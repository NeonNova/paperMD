# paperMD Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** A native macOS markdown editor/viewer (SwiftUI + NSTextView editor + WKWebView markdown-it preview), built and bundled entirely with SwiftPM/CLI ‚Äî no Xcode.

**Architecture:** MVVM. SwiftUI app shell with AppKit-wrapped `NSTextView` editor and a `WKWebView` preview running bundled markdown-it/KaTeX/Mermaid/highlight.js. Services own file I/O, autosave, session persistence, themes. Zero Swift package dependencies.

**Tech Stack:** Swift 6.0.3 toolchain (Swift 5 language mode), macOS 15 SDK / deployment target, SwiftPM executable target, vendored JS: markdown-it 14, markdown-it-footnote 4, markdown-it-task-lists 2, highlight.js 11, KaTeX 0.16, Mermaid 11.

**Commit policy (user preference):** ONE commit per task (feature-level). No micro-commits. Push to GitHub only at milestones (after Task 5 and after Task 10).

**Build/test commands:**
- Build: `swift build` (debug) ‚Äî run from repo root
- Test: `swift test`
- Run unbundled: `swift run`
- Bundle app: `Scripts/build-app.sh` (release build ‚Üí `dist/paperMD.app`)

- [ ] **Step 5.2: index.html + preview.js** ‚Äî single page, loads all vendor scripts + `preview.css` + theme CSS variables block. `preview.js` (complete logic):

```js
const md = window.markdownit({ html: false, linkify: true, breaks: false })
  .use(window.markdownitFootnote)
  .use(window.markdownitTaskLists, { enabled: false });

// wikilink: inline rule replacing [[Name]] / [[Name|Alias]] with styled span (not clickable)
md.inline.ruler.before('link', 'wikilink', (state, silent) => {
  const src = state.src, pos = state.pos;
  if (src.charCodeAt(pos) !== 0x5B || src.charCodeAt(pos + 1) !== 0x5B) return false;
  const end = src.indexOf(']]', pos + 2);
  if (end < 0) return false;
  if (!silent) {
    const inner = src.slice(pos + 2, end);
    const label = inner.includes('|') ? inner.split('|').pop() : inner;
    const t = state.push('wikilink', '', 0); t.content = label;
  }
  state.pos = end + 2; return true;
});
md.renderer.rules.wikilink = (tok, i) => `<span class="wikilink">${md.utils.escapeHtml(tok[i].content)}</span>`;

// code fences: hljs, except ```mermaid ‚Üí placeholder div with content hash
md.options.highlight = (str, lang) => {
  if (lang === 'mermaid') return null; // handled by fence override
  if (lang && hljs.getLanguage(lang)) return hljs.highlight(str, { language: lang }).value;
  return '';
};
const defaultFence = md.renderer.rules.fence;
md.renderer.rules.fence = (tokens, idx, opts, env, self) => {
  const t = tokens[idx];
  if (t.info.trim() === 'mermaid') {
    const h = hash(t.content);
    return `<div class="mermaid-block" data-hash="${h}" data-src="${encodeURIComponent(t.content)}"></div>`;
  }
  return defaultFence(tokens, idx, opts, env, self);
};

const mermaidCache = new Map(); let seq = 0;
async function renderMermaid() {
  for (const el of document.querySelectorAll('.mermaid-block')) {
    const h = el.dataset.hash;
    if (mermaidCache.has(h)) { el.innerHTML = mermaidCache.get(h); continue; }
    try {
      const { svg } = await mermaid.render('mm' + (seq++), decodeURIComponent(el.dataset.src));
      mermaidCache.set(h, svg); el.innerHTML = svg;
    } catch (e) { el.innerHTML = `<pre class="mermaid-error">${e.message}</pre>`; }
  }
}

window.renderMarkdown = function (text) {
  const y = window.scrollY;
  document.getElementById('content').innerHTML = md.render(text);
  renderMathInElement(document.getElementById('content'), {
    delimiters: [{left:'$$',right:'$$',display:true},{left:'$',right:'$',display:false}],
    throwOnError: false });

  window.scrollTo(0, y); // preserve scroll position
};
window.setThemeVars = function (vars) { // {background:'#‚Ä¶', ‚Ä¶} + fonts
